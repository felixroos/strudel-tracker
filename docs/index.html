strudel-tracker

This is an experiment to render a strudel pattern as a impulse tracker module. The conversion is done via itwriter. You can either download the converted .it file and open it in your favorite tracker or press "play" to listen to it in the browser. Playback via chiptune.js.

ctrl+enter = update+play, ctrl+. = stop

only sine and sawtooth are available as sounds so far
